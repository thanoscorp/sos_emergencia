# SOS Emergencia

Aplicacion de emergencia para identificar y comunicarse con victimas atrapadas bajo escombros mediante senales WiFi Direct.

## Que hace?

- **Senal WiFi Direct**: La victima activa el SOS y su telefono crea una red WiFi llamada `DIRECT-SOS-[DNI]`.
- **Drones de rescate**: Sobrevuelan la zona y detectan estas redes WiFi para localizar personas atrapadas.
- **Chat de emergencia**: Los rescatistas se conectan a la red y chatean con la victima via navegador web.
- **Mensajes de voz**: Walkie-Talkie integrado para cuando no se puede escribir.
- **Activacion automatica**: Se activa solo al perder senal celular.
- **Mensajes rapidos**: "Estoy herido", "Necesito agua", "Hay mas personas aqui".

## Como funciona (rescatista)

1. Busca la red WiFi: `DIRECT-SOS-XXXXXXXXXX`
2. Contrasena: `sos12345`
3. Abre en navegador: `http://192.168.49.1:8080`
4. Chatea directamente con la victima.
